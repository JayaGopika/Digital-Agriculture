Build a complete mobile application called "AgriBridge" that runs in Expo Go using React Native (Expo framework).

Use the provided AgriBridge logo in the splash screen and login page.

This app connects farmers directly with customers and also allows farmers to book cold storage for perishable goods.

Important Requirements

• The app must run in Expo Go
• Use React Native with Expo for frontend
• Use Node.js + Express for backend
• Use Firebase Firestore as the default database
• Use Firebase Authentication for login
• Include Speech to Text for product input
• Do NOT include voice control navigation
• Implement Push Notifications using Firebase Cloud Messaging
• Also send email notifications to Gmail when orders or bookings happen

App Roles

The application must support role-based login.

Roles:

• Farmer
• Customer
• Cold Storage Manager

After login, redirect users to the correct dashboard.
